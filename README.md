L-rensjon
=========

En applikasjon for å vise timeplanen til videregående skoler som bruker Nova Software sin Webviewer!

Kun støtte for Android foreløpig. Ta gjerne kontakt om du vil utvikle for andre plattformer!

Støttede skoler
 - Lørenskog VGS
 - Rælingen VGS
 - Strømmen VGS
 - Lillestrøm VGS
 - Mailand VGS
 - ++ Requests
